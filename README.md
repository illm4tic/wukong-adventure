# Wukong Adventure

[English](./README.md) | [简体中文](./README.zh-CN.md)

[Live Demo](https://wukong.yeezus.cn/)

`Wukong Adventure` is a fast-paced side-scrolling browser action game inspired by Sun Wukong. You fight through waves of enemies, build momentum with staff attacks, and push forward until the rift is fully cleared.

## About The Game

You play as Wukong in a compact combat stage built around quick movement, short attack windows, and simple but satisfying action. The goal is easy to understand:

- Defeat every enemy wave
- Stay alive while managing spacing and timing
- Advance to the end point after the final wave

## How To Play

The combat loop is built around basic pressure, mobility, and energy use:

- Use normal attacks to keep enemies under control
- Use the dash skill to break through pressure or close distance
- Save energy for the sweeping special attack
- Keep moving forward once a wave is cleared

## Controls

- `A / D`: Move left / right
- `W`: Jump
- `J`: Staff combo
- `K`: Somersault dash
- `L`: Ruyi sweep
- `P`: Pause / resume
- `Space`: Restart

## Run Locally

```bash
npm install
npm run dev
```

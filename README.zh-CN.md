# Wukong Adventure

[English](./README.md) | [简体中文](./README.zh-CN.md)

[在线体验](https://wukong.yeezus.cn/)

`Wukong Adventure` 是一个以孙悟空为主角的横版动作网页小游戏。玩家需要在裂隙战场中一路击退敌人、保持进攻节奏，并在清空所有波次后推进到终点。

## 游戏介绍

这款游戏主打的是直接、爽快的战斗体验：

- 扮演孙悟空进行闯关战斗
- 面对连续出现的敌人波次
- 在移动、跳跃、连击和技能之间寻找节奏

## 玩法说明

整体玩法很直接，重点是压制敌人并把握出手时机：

- 使用普通攻击持续输出并积累战斗节奏
- 用突进技能快速贴近敌人或脱离危险
- 积攒能量后释放范围技能清场
- 每一波清完后继续向终点推进

## 操作方式

- `A / D`：左右移动
- `W`：跳跃
- `J`：金箍棒连击
- `K`：筋斗突进
- `L`：如意横扫
- `P`：暂停 / 继续
- `Space`：重新开始

## 本地运行

```bash
npm install
npm run dev
```
